# Kotlin_Native

The `kotlin_native` commandlet allows to install and setup https://kotlinlang.org/[Kotlin_Native]. The arguments (`devon kotlin_native «args»`) are explained by the following table:

[options="header"]
|=======================
|*Argument(s)*                       |*Meaning*
|`setup`                             |setup Kotlin_Native (install or update and verify),  via `KOTLIN_NATIVE_VERSION` (e.g. `1.8.10`)
|`version`                           |print the version of the installed Kotlin-Native
|`version list`                      |list available Kotlin_Native versions
|`version set [«v»]`                 |set a specific Kotlin_Native version «v»
|=======================
